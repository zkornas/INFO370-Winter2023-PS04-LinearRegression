#6
m = smf.ols("GmSc ~ FG + FGA", harden_data).fit()
m.summary()
# a)
# The interpretation of FG is that for each field goal made the Game Score (GmSc) should increase by 3.0427 points.
# This is statistically significant as our t value is above 1.96 (8.383) and our p value is below 0.05 (0).
#
# b)
# The interpretation of FGA is that for each field goal attempt made, the Game Score will decrease by 0.5523 points, creating a negative relationship.
# This is statistically significant as our absolute t value is above 1.96 (|-2.553|), and our p value is below 0.05 (0.013).
#
# c)
# FGA in this model is negative (-0.5523), unlike in the previous model where FGA was positive (0.7037). This is due to our new model taking
# into account field goals made (FG), which would stay the same as FGA increases, meaning that they miss more and there for negatively
# affects the Game Score.
#
# d)
# The R^2 value is 0.594, meaning that 59% of variance in our Game Score (GmSc) can be explained by our predictor variables (FG + FGA). This is much
# higher than the previous model which only had an R^2 value of 0.133. This means that FG + FGA is a better predictor for variance in game score then
# just FGA alone.
